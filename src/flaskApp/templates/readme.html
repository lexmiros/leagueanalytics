Customer summoner name search

Searching for a summoner name on any region is implemented and fully functional.
However, the application currently only has a personal Riot Games API key.
This means that requesting data from the Riot games server is slow.
Data for up to 1000 games will be requested if available, with download times up to
20 minutes.

A production API key is under reqeust.

It is currently reccommended to use the pre-downloaded test data set.
This is achieved by submitting the home page form with an empty summoner name.
